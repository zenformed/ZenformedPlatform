## Upload Documents For a Workflow Task

### Overview

This article explains how to upload documents for workflow tasks within BuildCore.

### Step-by-Step

1. Open the project from the **Projects** view.
2. Navigate to the **Workflow Tasks** tab on the project page.
3. Locate the task that requires a document upload.
4. Click the attachment icon in the **Documents** column associated with that task.
5. From the dropdown menu, select **Upload** and choose the file you wish to attach.
6. The document will be linked to the task after uploading.
